# Платформа для онлайн-заказов еды

Веб-приложение на Flask для заказа еды из ресторанов с ролями: Администратор, Менеджер, Гость.

## Функциональность

- **Гость**: просмотр ресторанов и меню, добавление блюд в корзину, оформление заказа, просмотр истории заказов, отзывы.
- **Менеджер**: просмотр и изменение статусов заказов, управление отзывами.
- **Администратор**: полное управление ресторанами, блюдами и заказами.

## Технологии

- Python 3.10+
- Flask 2.3
- SQLAlchemy + SQLite
- Flask-Login (аутентификация)
- Bootstrap 5, Font Awesome

## Установка и запуск

1. Распакуйте архив, откройте проект внутри PyCharm.
2. **Создайте виртуальное окружение**:
   ```bash
   python -m venv venv

3. **Активируйте окружение:

    ```bash
    venv\Scripts\activate

4. Установите зависимости:

    ```bash
    pip install -r requirements.txt

5. Создайте базу данных и наполните тестовыми данными:
    Запустите файл init_data.py или в том же терминале среды разработки введите:

    ```bash
    python init_data.py

6. Запустите приложение:
    Запустите файл run.py или в терминале среды разработки введите:

    ```bash
    python run.py

7. Откройте браузер и перейдите по адресу http://127.0.0.1:5000.


## Примечание
Приложение использует SQLite (файл базы данных появится в папке instance/ после выполнения init_data.py).
